Pantalla o interfaces corridas

Este error ocurre cuando se desconfiguran los PPP (Puntos o Pixeles por Pulgadas). Para solucionarlo se debe realizar lo siguiente:

Solución para Windows 98 / Xp / Vista

  1. Inicio.
  2. Configuración.
  3. Panel de control.
  4. Pantalla.
  5. Configuración.
  6. Opciones Avanzadas y elegir la opción “fuentes pequeñas, o tamaño normal (96PPP)”.
  7. Iniciar el juego con normalidad.

Solución para Windows 7 u 8

  1. Inicio.
  2. Panel de control.
  3. Apariencia y personalización.
  4. Pantalla. En el menú de la izquierda debe elegir “Establecer tamaño de texto personalizado”. Debe mover la regla hasta que aparezca “96 pixeles por pulgada”. Luego debe hacer clic en “Aceptar” seguido de “Aplicar”.
  5. Iniciar el juego con normalidad.
